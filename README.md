# Infinitode 2 Hybrid Auto-Bot

This repository contains a fully automated, hybrid Python bot designed to play the tower defense game Infinitode 2. It utilizes a combination of computer vision for menu navigation and exact coordinate mapping for high-speed in-game actions. 

## 🚀 Features

* **Infinite Game Orchestration:** The `GameManager` automatically handles level selection, game initialization, active gameplay, and post-game reward screens in an endless loop.
* **Computer Vision Controller:** Scans the active game region to locate key UI elements (e.g., menus, play buttons) dynamically.
* **High-Speed Hotkey Interaction:** The `FastEntity` system uses absolute coordinate math and keyboard hotkeys (like `n` for Crushers or `3` for Green Miners) to build and upgrade units instantly.
* **Automated Run Logging:** Successfully completed games are automatically appended to a `games_played.log` file with timestamps.
* **Coordinate Mapping Utility:** Includes a dedicated script to help you easily map relative `x` and `y` coordinates for different tiles and UI elements without guessing.
* **Dual Fail-Safes:** Includes PyAutoGUI's corner fail-safe and a hard-coded `q` keyboard kill switch to abort operations safely.

The scripts require the following Python libraries:
* `pyautogui`
* `pygetwindow`
* `keyboard`
* `opencv-python` / `Pillow` (Implicitly required for PyAutoGUI's vision features)

## 📂 Project Structure

* `main_bot.py`: The core bot script containing the `GameManager`, `VisionController`, and `FastEntity` classes.
* `mapper.py`: A command-line utility used to capture and log relative screen coordinates for new custom map setups.
* `mapped_coordinates.txt`: The output log file generated by the mapper script, storing captured absolute and relative coordinates for game entities like miners and speed cards.
* `images/`: A required local directory that must contain UI reference images for the bot to function.

## 🛠️ Setup & Usage

### 1. Prepare the Image Directory
To navigate menus, the bot requires specific image snips saved in an `images/` directory. Ensure you have screenshots of the following buttons:
* `select_level.png`.
* `level_6_3.png`
* `continue.png`
* `play.png`
* `start_wave.png`

### 2. Mapping Coordinates (Optional)
If you need to adjust where the bot clicks for specific towers or buttons:
* Ensure Infinitode 2 is open.
* Run the mapper script.
* Hover your mouse over the desired tile or button.
* Press `SPACE` to calculate and save the relative coordinates to `mapped_coordinates.txt`.
* Press `Q` to quit the mapper.

### 3. Running the Auto-Bot
* Run the main bot script.
* The script will automatically search for the "Infinitode 2" window, restore it if minimized, and lock onto its screen region.
* It will execute in four distinct phases: Start Round Setup, Game Init Setup, Gameplay Loop, and Game End Setup.
* **To stop the bot at any time:** Press and hold `q` to trigger the manual kill switch.
